# CIRCL pyecsca-style page

This repository is a small static site that presents a pyecsca-style library analysis page for Cloudflare CIRCL, focusing on the `dh/x25519` and `dh/x448` submodules.

## Files

- `index.html` – redirect entry point
- `circl.html` – main page
- `style.css` – lightweight Sphinx/Alabaster-like styling

## GitHub Pages

1. Create a new repository on GitHub.
2. Upload these files to the repository root.
3. In repository settings, enable **GitHub Pages** from the main branch root.
4. Open the published site URL.

## Notes

The page is intentionally styled to resemble a pyecsca library page while remaining an original static HTML implementation.
